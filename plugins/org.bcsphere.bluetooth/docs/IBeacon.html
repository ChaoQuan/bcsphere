Class: IBeacon

IBeacon

new IBeacon(iBeaconID, txPower, RSSI, accuracy, proximity)

IBeacon is the BLE device which matches the Apple iBeacon format.
Parameters:
Name Type Description
iBeaconID string The iBeaconID include all info about this iBeacon(proximityUUID/major/minor)
txPower int The iBeacon txPower
RSSI int The RSSI
accuracy float The accuracy of this iBeacon
proximity float The proximity of this iBeacon
Properties:
Name Type Description
proximityUUID string The proximityUUID of this iBeacon
major string The major number of this iBeacon
minor string The minor number of this iBeacon
accuracy float The accuracy of this iBeacon
proximity int The proximity of this iBeacon(0:Unknown | 1:Less than half a meter away | 2:More than half a meter away, but less than four meters away | 3:More than four meters away)
